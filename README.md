# Compiler Project

## Milestone 1 : Scanner
In this we have constructed a scanner of source language to output tokens in a tabular form


### Steps to Build and Run 

```
$ cd /src
$ make
$ cd ..
$ ./bin/lexer /test/test1.c

```
